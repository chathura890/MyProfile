## ğŸ› ï¸ Installation & Setup

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Git installed on your system
- Code editor (VS Code recommended)

### Local Development Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/chathura890/MyProfile.git
   cd MyProfile
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - Or use a local server (recommended):
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (if you have it)
   npx serve .
   ```

3. **View the website**
   - Open http://localhost:8000 in your browser
   - The website should load with all features working

### Customization

1. **Update Personal Information**
   - Edit `index.html` - Update name, description, contact details
   - Replace profile image in `images/` folder
   - Update social media links

2. **Modify Styling**
   - Edit `styles.css` - Change colors, fonts, layout
   - Update CSS custom properties in `:root` for theme changes

3. **Add Functionality**
   - Edit `script.js` - Add new interactive features
   - All JavaScript is vanilla - no framework dependencies
